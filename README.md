## Datum - Responsive Admin Dashboard Template

![screenshot](/assets/images/screenshot.png){width=640 height=480}

### **Overview**

-   **Name:** Datum - Responsive Admin Dashboard Template
-   **Purpose:** A pre-built admin dashboard template with various UI components, charts, tables, and forms.
-   **Features:**
    -   Sidebar navigation with multiple sections (Dashboard, UI Kits, Forms, Tables, Charts, Maps, etc.).
    -   Multiple charting and data visualization tools (D3.js, NVD3, Flot, Chart.js).
    -   Various layout configurations (Boxed, Fixed Sidebar/Header, Collapsed Sidebar).
    -   Integrated chat and notifications system.
    -   Support for different form elements, including file uploads and image cropping.
    -   Custom themes and CSS modifications via `Datum.min.css` and `custom.css`.
    -   Mobile-responsive design using Bootstrap.

### **Frameworks and Libraries Used in the Application**

#### **CSS Frameworks**

1.  **Bootstrap** (`assets/plugins/bootstrap/css/bootstrap.min.css`)
    
    -   Used for responsive design, grid system, and UI components.
2.  **Font Awesome** (`assets/plugins/font-awesome/css/font-awesome.min.css`)
    
    -   Provides scalable vector icons and social media logos.
3.  **Icomoon** (`assets/plugins/icomoon/style.css`)
    
    -   Custom icon font set for additional UI elements.
4.  **Uniform.js** (`assets/plugins/uniform/css/default.css`)
    
    -   Enhances form elements for consistent cross-browser styling.
5.  **Switchery.js** (`assets/plugins/switchery/switchery.min.css`)
    
    -   Adds toggle switch UI components.
6.  **NVD3** (`assets/plugins/nvd3/nv.d3.min.css`)
    
    -   Styling for D3.js-based charts.

___

#### **JavaScript Libraries and Plugins**

1.  **jQuery** (`assets/plugins/jquery/jquery-3.1.0.min.js`)
    
    -   Core JavaScript library for DOM manipulation and AJAX requests.
2.  **Bootstrap JavaScript** (`assets/plugins/bootstrap/js/bootstrap.min.js`)
    
    -   Enables Bootstrap’s interactive components (modals, tooltips, etc.).
3.  **SlimScroll.js** (`assets/plugins/jquery-slimscroll/jquery.slimscroll.min.js`)
    
    -   Used for custom scrollbars.
4.  **Uniform.js** (`assets/plugins/uniform/js/jquery.uniform.standalone.js`)
    
    -   Enhances form elements styling.
5.  **Switchery.js** (`assets/plugins/switchery/switchery.min.js`)
    
    -   Converts checkboxes into iOS-style switches.
6.  **D3.js** (`assets/plugins/d3/d3.min.js`)
    
    -   A JavaScript library for data visualization.
7.  **NVD3.js** (`assets/plugins/nvd3/nv.d3.min.js`)
    
    -   A wrapper for D3.js used for charting.
8.  **Flot Charts**
    
    -   (`assets/plugins/flot/jquery.flot.min.js`)
    -   (`assets/plugins/flot/jquery.flot.time.min.js`)
    -   (`assets/plugins/flot/jquery.flot.symbol.min.js`)
    -   (`assets/plugins/flot/jquery.flot.resize.min.js`)
    -   (`assets/plugins/flot/jquery.flot.tooltip.min.js`)
    -   (`assets/plugins/flot/jquery.flot.pie.min.js`)
    -   A lightweight JavaScript plotting library.
9.  **Chart.js** (`assets/plugins/chartjs/chart.min.js`)
    
    -   Another charting library for creating beautiful charts.
10.  **Custom JavaScript Files**
    
    -   `assets/js/Datum.min.js` → Minified core JavaScript for the template.
    -   `assets/js/pages/dashboard.js` → Handles dashboard-specific functionalities.
